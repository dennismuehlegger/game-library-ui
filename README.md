# Game Library UI

Frontend for the [Game Library](https://github.com/dennismuehlegger/game-library) REST API. Built with vanilla HTML, CSS, and JavaScript.

---

## Overview

A simple browser-based interface for the Game Library backend. Users can browse available games and purchase them through the existing API.

---

## Features
- Browse games with cover art
- Purchase games as a selected user
- Toast notifications for purchase outcomes
- User library view
- Transaction history
- Multiple tabs for marketplace, library, and history

---

## How to Run

1. Make sure the [Game Library backend](https://github.com/dennismuehlegger/game-library) is running on `http://localhost:8080`
2. Open `index.html` in your browser

No build tools or dependencies required.
